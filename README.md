# Contact Manager App

Architecture, Design Pattern and Principles :
1) Clean/Onion/Hexagonal Architecture
2) Dependency Injection
3) SOLID Principles
4) DRY Principle
5) KISS Principle

Technology :
1) Asp.Net Core 2.1
2) Angular 8
3) SQL Server 2017
4) Boostrap 4
